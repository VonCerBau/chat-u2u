# Chat (u2u)

## Justification

- Frontend and backend development
- Realtime messaging communication
- Containerization
- Easy deployment

## Run

### Start

```bash
docker compose up -d
```

### Access

```bash
http://localhost:3000
```

### Stop

```bash
docker compose down
```
